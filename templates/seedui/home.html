Register seeds from six images and identify them from one.

This interface wraps the simplified image pipeline: users upload six directional photos to build a stored seed profile, and later upload a single query image to retrieve the closest matches.

Registration Flow

  1. Upload front, back, left, right, top, and bottom images.
  2. Enter a seed name and optional source notes.
  3. The system extracts silhouettes, calculates parameters, and stores a voxel-style profile.

Identification Flow

  1. Upload a single seed image from any visible side.
  2. The system compares it with all saved training views and generated projections.
  3. The UI shows ranked matches with similarity scores and the best matching view.
